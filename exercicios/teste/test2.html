Príncipais formatações

Negrito/Destaque

Para por termos em destaque usa-se a tag "stong"

Itálico/Ênfase

Para por um texto em itálico usa-se a tag "em"

Texto marcado

Para marcar um texto usa-se a tag MARK
Além disso é possível mudar a cor usando css. Dentro da abertora da tag MARK use a tag style="background-color:;" e escolha uma cor

Para ter todas as tags MARK padronizadas é preciso criar uma tag "style" no "head" do HTML e fazer um seletor para "mark". Dessa forma, a cor selecinada no seletor vale como cor padrão da tag "mark"
