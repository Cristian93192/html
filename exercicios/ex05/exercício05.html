Formatações

Principais Formatações

Negrito/Destaque

Usa-se a tag <strong>

Itálico/Ênfase

Usa-se a tag <em>

Texto Marcado

Usa-se a tag <mark>
Para mudar a cor:
<mark style="background-color: 000000;"></mark>
Para alterar a cor de todos os "marks" é preciso criar uma tag "style" no "head" e fazer um seletor para "mark". Ex: 

				<style>
					mark{backgroud-color:red;}
				</style>

Formatações Adicionais

Texto grande e pequeno

Para texto grande usa-se a tag <big>(Tag obsoleta)
Para texto pequeno usa-se a tag <small>

Texto deletado

Indica que o texto deve ser lido mas não considerado
Usa-se a tag <del>
Ex: Texto excluido

Texto inserido

Dá ênfase e indica que foi adicionado depois
Usa-se a tag <ins>
Ex: Texto inserido

Texto sobrescrito

Usa-se a tag <sup>
Ex: x20+3

Texto subscrito

Usa-se a tag <sub>
Ex: H2O

Outras Formatações

Código-fonte

Usado para fazer todas as letras ocuparem o mesmo espaço
Esa-se a tag <code>
Ex: Código-fonte

Pré-formatação

Usado para que a forma final fique igual a do código
Ex: 

Pré		-		Formatação
			000
		0000			0000

Citações simples

Põe a frase entre aspas("")
Usa-se a tag <q>

Citações completas

Cria uma citação deslocada e permite a criação de um parâmetro "cite" para colocar o endereço de link da citação
Usa-se a tag <blockquote>
Ex:

Esquecer é uma necessidade. A vida é uma lousa, em que o destino, para escrever um novo caso, precisa de apagar o caso escrito

Abreviações

Permite colocar o significado de uma sigla através de um pequeno pop-up
Usa-se a tag: <abbr>
Ex: HTML

Texto invertido(meio inútil)

<bdo dir="rtl">Inverte o texto</bdo>
<bdo dir="ltr">Não inverte</bdo>
