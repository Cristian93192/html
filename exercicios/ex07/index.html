Usando links externos

Você pode adicionar links externos em uma parte do texto usando a tag <a>. Essa tag permite alguns parâmetros:

  • href - para por o link
  • target-="_blank" - para abrir o link em uma nova aba
  • rel="external" - indica que o link não faz parte do seu site

Ex: Você pode acessar o repositório do GitHub
